# zcpy
中文程式(無聊做的)

# 使用
## 基本語法
- 用[ ]框起來的表示變數,實際使用時不需要添加
- 必須在每具後面加上 `,` 在判斷式結束時要加 `.`
```txt
顯示 [內容],
輸入 [變數],
如果 [內容] 等於 [內容],
如果 [內容] 不等於 [內容],
```
- 如果語法
```txt
如果 [內容] 等於 [內容],
顯示 動作1,
顯示 動作結束.
```
